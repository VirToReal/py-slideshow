**Requirements**:

* Python 2.7+ or 3.2+
* pyglet

**Command Line Help**::

    $ ./slideshow.py -h
    usage: slideshow.py dir

    positional arguments:
      dir              directory of images

**Example**::

    $ git clone https://github.com/jamincollins/py-slideshow.git
    $ cd py-slideshow
    $ python slideshow.py images/
